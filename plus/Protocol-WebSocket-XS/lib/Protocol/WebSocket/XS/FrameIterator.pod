=head1 NAME

Protocol::WebSocket::XS::FrameIterator - WebSocket frame iterator class

=cut

=head1 DESCRIPTION

The class allows developer traverse on parsed websocket frames.

You cannot instantiate an object of the class direcly, it can be accessed
only via L<Protocol::WebSocket::XS::Parser> or
L<Protocol::WebSocket::XS::MessageIterator>.


=head1 METHODS

=head2 next()

Returns next unvisited L<Protocol::WebSocket::XS::Frame> parsed from receive buffer;
otherwise (i.e. when end of buffer is reached) returns C<undef>.

=head2 get_messages()

Returns L<Protocol::WebSocket::XS::MessageIterator>, if there is any unread frame left.
Otherwise returns C<undef>.
